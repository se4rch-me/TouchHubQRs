# **App Name**: Touch Center Payment Hub

## Core Features:

- Payment Method Grid: Displays a grid layout optimized for iPads, showcasing different payment options like Nequi, Daviplata, and bank accounts, each with its dedicated QR code and account details.
- QR Code Display: Clearly displays QR codes for each payment method (Nequi, Daviplata, banks) within their respective cards.
- Account Info Display: Displays account details (bank name, account type, account number) for direct bank transfers with the possibility to include a QR code.
- Copy to Clipboard: Provides a 'Copy' button next to each account number or phone number for easy copying, with visual feedback upon successful copy.
- Mobile Optimization: Ensures the page is optimized for iPad displays using viewport meta tags and disabling user scaling to prevent accidental zooming.
- Loading Animation: Implements a subtle fade-in animation for payment cards on page load to enhance user experience.

## Style Guidelines:

- Background color: Pure white (#FFFFFF) to facilitate QR code scanning.
- Primary color: Dark grey (#333333) for text to ensure readability.
- Accent color: Soft blue (#A0B4B7) to subtly highlight different payment options, inspired by minimalist financial interfaces.
- Font: 'Inter' (sans-serif) for a clean, modern, professional look suitable for both headlines and body text. Note: currently only Google Fonts are supported.
- Grid system: Use a 2-column grid for iPads to display payment options.
- Use Lucide Icons for bank icons and copy functionality, ensuring they are visually consistent and professional.
- Fade-in animation for payment cards to enhance the loading experience.
